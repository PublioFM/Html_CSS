As Tags do HTML5

Em HTML5 utilizamos Tags para executar comendos específicos. Para inserir cada um dos itens abaixo utiliza-se um tipo de TAG. Passe o mouse sobre a palavra e descubra qual TAG foi utilizada.

  • Parágrafos
  • Titulos
  • Imagem
  • Abreviações
  • Lista numerada
  • Lista de marcação
  • Hyperlinks
