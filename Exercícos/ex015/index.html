Capítulo 1

Esta página está configurada com o que chamamos de CSS locais ou internas. Com essa técnica é mais vantajosa do que a CSS Iline. Ela deixa o código HTML mais organizado e estéticamente mais bonito. Iremos utilizá-la bastante no dia a dia para fazer configarções locais em páginas simples que não serão aplicadas em outros documentos.
Mas ela ainda tem duas desvantagens:
1ª - O estilo (style), ocupa muitas linhas, na maioria das vezes fica moir do que o próprio conteúdo (HTML).
2ª - Se for um site com muitas páginas os estilos terão que ser configurados de uma em uma.
Para suprir essas necessidades citadas acima iremos aprender um outro método nas próximas aulas, que é o CSS Externo.
