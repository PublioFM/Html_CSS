Principais formatções

Negrito / Destaque

Nessa frase temos um termo em negrito usando a TAG <b> (não semântica).

Nessa outra temos um termo em destaque usando a TAG <strong> (semântica).

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a TAG <i> não semântica).

Nesta frase temos um termo em ênfase usando a TAG <em> (semântica).

Texto Marcado

Podemos criar tambem um texto marcado usando a TAG <mark> (semântica).

E no outro parágrafo temos outro texto marcado, também usando a TAG <mark> (semântica).

Essa diferença de cor nas marcações do texto foram feitas utilizando a TAG PARÂMETRO <style>, do CSS, dentro da TAG <mark>.

Texto Grande e Pequeno

Estamos criando, neste parágrafo, um texto grande utilizando a TAG <big>, já obsoleta (não semântica), e um texto pequeno utilizando a TAG <small> (semântica).

Texto deletado

Podemos marcar um texto como deletado, para indicar que ele deve ser lido, mas não considerado, utilizando a TAG <del> (semântica).

Texto inserido

Podemos marcar um texto como inserido, para dar uma ênfase e indicar que ele foi adicionado depois, utilizando a TAG <ins> (semântica).

Existe também o sublinhado com a tag <u>(não semântica).

Texto Sobrescrito

Para inserir informações do tipo x3 + 5 = 13, utilizando a TAG <sup> (semântica).

Texto Subscrito

Para inserir informações do tipo H2O, utilizando a TAG <sub> (semântica).
