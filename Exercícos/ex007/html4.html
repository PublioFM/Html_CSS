Exemplo de site em HTML4

Essas TAGS em vermelho (no VS Code), significam que são TAGS obsoletas. Elas podem ainda funcionar, mas em breve serão desativadas. Eram utilizadas para se desenolver em HTML4.

O HTML5 é focado em semântica, não em forma. Ou seja, ele é feito para se desenolver utilizando o significado (semântica) das TAGS para resultar em conteúdo para os sites.


Para dar forma, ou tornar o site mais bonito e agradável visualmente, utiliza-se o CSS3.
